# Plain English Architecture Requests
# Examples that non-technical people can write

---

## Example 1: Simple Business Website

I need a website for my small business.

Requirements:
- Customers can visit from anywhere
- Contact form that saves to database
- Photo gallery of our work
- Need it to be secure and backed up
- Should work even if server fails
- Want to know if website goes down

---

## Example 2: Mobile App Backend

We're building a mobile app and need the backend infrastructure.

What we need:
- API for mobile app to connect to
- Database to store user accounts and app data
- Storage for user uploaded photos
- Push notifications
- User login and authentication
- Fast and scalable (expecting 10,000 users)
- Secure - encrypt user data
- Monitoring so we know if something breaks

# How to use these examples:

Just copy one of these examples, customize it for your needs, save as a .txt file, and run:

```bash
/draw-aws-diagram /path/to/your-requirements.txt
```

The AI will automatically figure out you need:
- VPC and networking
- Load balancers and servers
- Databases
- Storage (S3)
- Monitoring (CloudWatch)
- Security (KMS, Secrets Manager)
- And more!

No need to know AWS services - just describe what your application needs to do!
